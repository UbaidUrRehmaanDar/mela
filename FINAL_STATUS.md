## 🚀 Next Steps

### 1. Deploy Firebase Backend
```bash
cd backend
firebase login
firebase deploy
```

### 2. Run Frontend
```bash
cd frontend
npm run dev
```

### 3. Create First Moderator
- Sign up in app
- Go to Firestore Console
- Edit user document:
  ```json
  {
    "role": "moderator",
    "moderatorFor": ["FAST NUCES"]
  }
  ```
